This is a project developing a tiny control board used on small rockets.

FEATURES |  STM32F103C8T6 mcu,72Mhz,64KB flash;
            MPU6050 gyro with dmp attitude solution;
            FreeRTOS; 
            4 CH servo control;
            1 CH USART;
            2 CH MOS control power stage output;
            6V-12V power input;
